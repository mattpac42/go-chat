# Garden Request Command

Generate an improvement report for The Garden team.

## Usage

```
/garden-request
```

## What It Does

Creates a structured report in `.claude/work/garden-requests/` that can be sent to The Garden maintainers for bugs, features, or workflow improvements that should be upstreamed.

## Interactive Flow

1. Ask for title and category
2. Gather current vs proposed behavior
3. Generate formatted report with GR-XXX ID
4. Provide submission instructions
